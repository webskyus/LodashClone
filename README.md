# LodashClone
A modern JavaScript utility library delivering modularity, performance &amp; extras.
__________________________________
#### ✅ Array / ._chunk
<code>
_.chunk(['a', 'b', 'c', 'd'], 2);
// => [['a', 'b'], ['c', 'd']]
</code>

__________________________________
#### ✅ Array / ._compact
<code>
_.compact([0, 1, false, 2, '', 3]) => [1, 2, 3]
</code>
